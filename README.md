# mag: eMail-Address Getter

Webブラウザでテキストを選択(Ctr;+A)して、コピー(Ctrl+C)する事で、ページに含まれるメールアドレスを列挙します。  

## 開発環境

- Microsoft Visual Basic.NET 2005 で作成しました。
- Windows 10/11 で動作確認していません。
